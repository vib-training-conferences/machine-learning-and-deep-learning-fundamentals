# Machine Learning and Deep Learning Fundamentals

Welcome to the repository for the **Machine Learning and Deep Learning Fundamentals** workshop, organised by **VIB**.

This repository contains the materials for the interactive course, which is built using [LiaScript](https://liascript.github.io), a platform for creating interactive educational resources.

---

## Course Overview

- **Level:** Beginner
- **Target Audience:** Researchers, training providers
- **Duration:** 2 days
- **Prerequisites:** Familiarity with Python (no prior machine learning experience required)

### Schedule (summary)

**Day 1**
- Introduction to machine learning
- Regression
- Classification
- Model selection

**Day 2**
- Introduction to neural networks
- Deep neural networks
- Convolutional neural networks
- Advanced deep learning topics
